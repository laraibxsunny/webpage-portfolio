Asteroid Shooter

I started this project to introduce myself into developing, with two versions both using Python with Pygame. My first iteration showed a basic understanding of the language using mostly ‘for loops’, but my second iteration exhibited much better comprehension of Python by using classes, inheritance and maths components to make the code more reusable and efficient. Asteroid shooter trained me to be resilient and strong headed when I felt like the code was against me.
